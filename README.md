PHP API для работы с НПД (Налог на профессиональный доход)
==========================================================

Этот репозиторий содержит PHP-скрипты для взаимодействия с API [lknpd.nalog.ru](https://lknpd.nalog.ru/), включая аутентификацию, создание доходов, получение профиля и списка доходов.

📂 Файлы
--------

*   `create_income.php` - создание нового дохода (чека).
*   `get_profile.php` - получение информации о налогоплательщике.
*   `incomes.php` - получение списка доходов.

🚀 Установка и использование
----------------------------

1\. Склонируйте репозиторий:
   `git clone https://github.com/your-repo/npd-api.git`

2. Разместите файлы на сервере с поддержкой PHP.

3. Убедитесь, что на сервере установлен cURL.

4. Отправляйте POST-запросы к соответствующим скриптам.

🔧 Пример использования
-----------------------

**Запрос к create\_income.php:**
    
    curl -X POST https://your-server.com/create\_income.php \\
         -H "Content-Type: application/json" \\
         -d '{
              "username": "ваш\_логин",
              "password": "ваш\_пароль",
              "amount": 1000,
              "comment": "Оплата услуг"
            }'

⚠️ Важно
--------

*   Используйте защищённые соединения (HTTPS) при передаче данных.
*   Не храните логины и пароли в коде, используйте переменные окружения.
